# Handling CUT&Tag with diffrent organism(s) spike-in
Measurement and utilization of spike-in variation and proportion in CAT&Tag data, QC, etc.

| | Folder | Content |
| --- | ------------- | ------------ |
| 0. | `mapping_and_peak_calling/` | bash functions: mapping on a chimeric genome, peak calling | 
| 1. | `stats_and_QC/` | R and bash funcitons: QC plots, fragment lengths distributions, organisms and chromosomes proportions, etc |
| 2. | `bw_adjustment/` | snakemake and bash functions: computing of scaling factors from spike-in proportions, and scaling of bw files |
